# LiftLedger Backend

This is the backend service for LiftLedger applications.

## 🚀 Features
- Create workouts

## 🗺️ API Routes
| Method | Endpoint | Description
| --- | --- | --- |
| `GET`  | `/health` | Checks the status of the server |
| `GET`  | `/workouts/{id}` | Retrieve a workout by ID |
| `POST` | `/workouts` | Create a workout |
